# fletflow

Technologies:

* [Flet](https://flet.dev/)
* [Copier](https://copier.readthedocs.io/)

Getting started:

1. Create a directory for your project
2. Within it, run `copier copy git@github.com:gaboverse/fletflow.git .`
3. Create your repo
4. Run `git init` and add the remote.
5. Add, commit, push... and Enjoy!
